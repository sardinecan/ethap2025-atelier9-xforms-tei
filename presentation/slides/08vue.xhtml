La vue - les contrôles simples

XForms propose plusieurs contrôles de formulaire pour interagir avec les données

  • champ texte (<xf:input/>) ;
  • zone de texte (<xf:textarea/>) ;
  • liste à choix unique (<xf:select1/>) ;
  • case à cocher (<xf:select/>) ;
  • label (<xf:label/>) ;
  • groupe contextuel (<xf:group/>)
  • sortie textuelle (<xf:output/>) ;
  • bouton (<xf:trigger/>)

Ces contrôles sont reliés à une instance par l'intermédiaire d'un attribut @ref prenant pour valeur un chemin XPath.
